时钟周期

xxxxxxxxxx
时钟周期：
    是处理操作的最基本单位；
    通常称为 节拍脉冲或 T 周期；

机器周期

xxxxxxxxxx
机器周期：
    也称为 CPU 周期；
    一条指令的执行过程划分为若干个阶段（如取指、译码、执行等），每一个阶段完成一个基本操作，完成一个基本操作所需要的时间称为机器周期；
    常用从内存中读取一个指令字的最短时间来规定 CPU 周期；
    一个机器周期由若干个状态周期组成；

指令周期

xxxxxxxxxx
指令周期：
    执行一条指令所需要的时间，一般由若干个机器周期组成；
    指令不同，所需的机器周期数也不同，常常用若干个 CPU 周期数来表示；
    一条指令的取出阶段需要一个 CPU 周期时间，而一个 CPU 周期时间又包含若干个时钟周期，这些时钟周期的总和，决定了一个 CPU 周期的时间宽度；
    取出和执行任何一条指令所需的最短时间为两个 CPU 周期；
    对于复杂一些的指令，则需要更多的 CPU 周期；

存储周期

xxxxxxxxxx
存储周期：
    指主存储器两次启动操作之间需要的最小时间间隔，也称之为主存储器周期时间，稍长于主存读写存取周期；
    CPU 通过使用AR（地址寄存器）、DR（数据缓冲寄存器）、总线与主存进行数据传送；
    连续完成两次读/写操作中间需待前一操作信号全部消失后才能进行下一个操作，故半导体存储器的周期时间通常稍大于其存取时间，而磁芯存储器的周期时间通常是其存取时间的两倍；

总线周期

xxxxxxxxxx
总线周期：
    主存和I/O端口是挂接在总线上的，CPU对主存和I/O接口的访问是通过总线实现的；
    CPU 通过总线对微处理器外部的主存或I/O接口进行一次访问所需的时间成为一个总线周期；
    一个总线周期一般包含 4 个时钟周期，分别称为 4 个状态（T1状态、T2状态、T3状态、T4状态）；

CMMI成熟度模型
1.阶段式模型
阶段式模型的结构类似于 CMM,它关注组织的成熟度。CMMI有5个成熟度等级:
    初始的:过程不可预测且缺乏控制。
    已管理的:过程为项目服务。
    已定义的:过程为组织服务。
    定量管理的:过程已度量和控制。
    优化的:集中于过程改进。
2.连续式模型
连续式模型关注每个过程域的能力，一个组织对不同的过程域可以达到不同的过程域能力等级(Capability Leve, CL)。CMMI中包括6个过程域能力等级，能力等级可以独立地应用于任何单独的过程域，任何一个能力等级都必须满足比它等级低的能力等级的所有准则。对各能力等级的含义简述如下:
    CLO(未完成的):过程域未执行或未得到CL1中定义的所有目标。
    CL1(已执行的):其共性目标是过程将可标识的输入工作产品转换成可标识的输出工作产品，以买现支持过程域的特定目标。（本答案）
    CL2(已管理的):其共性目标集中于已管理的过程的制度化。根据组织级政策规定过程的运作将使用哪个过程，项目遵循文档化的计划和过程描述，所有正在工作的人都有权使用足够的资源，所有工作任务和工作产品都被监控、控制和评审。
    CL3(己定义级的):其共性目标集中于已定义的过程的制度化。过程是按照组织的剪裁指南从组织的标准过程集中剪裁得到的，还必须收集过程资产和过程的度量，并用于将来对过程的改进。
    CL4(定量管理的):其共性目标集中于可定量管理的过程的制度化。使用测量和质量保证来控制和改进过程域,建立和使用关于质量和过程执行的定量目标作为管理准则。
    CL5(优化的):使用量化(统计学)手段改变和优化过程域，以满足客户要求的改变和持续改进计划中的过程域的功效。

使用ADSL接入Internet，用户端需要安装 PPPoE协议；
PPPoE是利用以太网发送PPP并且支持在同一以太网上建立多个PPP连接的接入技术。其结合了以太网和PPP连接的综合属性。在ADSL拨号中，经常应用PPPoE，用于向广大普通用户提供认证、计费服务，也可用于固定用户申请独立的一个公网IP地址。PPPoE认证的主要特点在于其应用于广泛、成熟；而且标准性、互通性好。
